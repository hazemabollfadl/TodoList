# Todo-List

Todo-List is a client based app similar to Notes on iPhone, the popular Note taking app that was Devoloped by Apple.
a service called Realm Database as a backend database to store and retrieve your Data from the iPhone Storage. 

## Functionality

* integrated third party libraries in the app using Cocoapods.
* stored data in the iPhone storage using Realm database.
* Smooth Navigation through the APP.
* Search items in a list.
* Swipe to delete.
* Easily Add Categories and their associated Items.
